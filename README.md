# Colon D

Colon D is a small macOS menu-bar app for typing emoji by shortcode. Type a colon-prefixed query, pick the emoji you want, and Colon D replaces the typed trigger with the selected emoji in the active text field.

The app is designed as a quiet utility: it lives in the menu bar, runs globally when enabled, and works across apps that expose enough macOS accessibility information for text replacement.
